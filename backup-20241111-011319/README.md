## Project Structure

│   └── src/

│   └── src/

## Features
1. Notebook Organization
   - Notebooks and sections
   - Hierarchical note structure
   - Drag and drop organization

2. Rich Text Editing
   - Text formatting (bold, italic, underline)
   - Lists and tables
   - Code blocks
   - Headings and styles

3. Real-time Saving
   - Autosave functionality
   - Version history
   - Offline support

4. File Attachments
   - Image upload and display
   - File attachments
   - Drag and drop support

5. Tags and Search
   - Note tagging system
   - Full-text search
   - Filter by tags
